# Uber-E
Use Flask to implement a Food Ordering Service on localhost

## Version
    >  python 3.7
    >  MySQL  8.0

## Installation
 
    $ pip install flask
    
in order to encrypt password by hash function

    $ pip install flask_bcrypt
